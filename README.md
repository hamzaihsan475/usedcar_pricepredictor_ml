\# 🚗 Car Price Predictor



A machine learning project that predicts used car prices using Linear Regression.



\## 📁 Project Structure

usedcar_pricepredictor_ml/

├── data/

├── model/

├── api/

├── frontend/

├── requirements.txt

└── README.md



\## 🤖 Model Performance

\- Algorithm: Linear Regression

\- R² Score: 0.89

\- MAE: ₹1,36,469



\## 🛠️ Tech Stack

\- Python, Scikit-learn

\- FastAPI

\- Streamlit

\- Pandas, NumPy



\## 🚀 How to Run



\### Start API

cd api

uvicorn main:app --reload



\### Start Frontend

cd frontend

streamlit run app.py
